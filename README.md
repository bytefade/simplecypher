# simplecypher
Roteiro:
- Cifrar texto simples em português sem acento OK
- Cifrar números OK
- Criar seperador de palavras OK
- Decifrar texto OK
- Melhorar a interface
- Cifrar/Decifrar texto com acentos
- Cifrar/Decifrar texto com pontuação, caracteres especiais
- Permitir configurar a forma de cifragem (não salvar em banco sql)
- Permitir baixar as configurações de cifragem
- Peritir subir as configurações de cifragem
